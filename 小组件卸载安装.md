可以的，用 **winget** 命令重新安装回来：

## 方法一：winget 安装（推荐）

打开**终端（管理员）** 或 **PowerShell（管理员）**，输入：

```
winget install "Windows Web Experience Pack"
```

或者用包名更精确地安装：

```
winget install 9MSSGKG348SP
```

`9MSSGKG348SP` 是它在微软商店的产品 ID，更可靠。

## 方法二：从微软商店重新获取

```
start ms-windows-store://pdp/?productid=9MSSGKG348SP
```

这条命令会直接打开微软商店中小组件的页面，点击安装即可。

------

安装完成后，如果任务栏没有自动出现天气图标，手动开启一下：

1. 右键任务栏 → **任务栏设置**
2. 找到"**小组件**"，打开开关

重启资源管理器或重启电脑后一定会生效。
